\section{Troubleshooting Guide}

\subsection{Common Error Messages}

\subsubsection{MongoDB Connection Error}
\textbf{Error:} \texttt{MongoNetworkError: connect ECONNREFUSED}
\textbf{Solution:} Ensure MongoDB is running and connection URI is correct

\subsubsection{Port Already in Use}
\textbf{Error:} \texttt{Error: listen EADDRINUSE: address already in use :::5000}
\textbf{Solution:} Kill the process using the port or change port in .env

\subsubsection{CORS Error}
\textbf{Error:} \texttt{Access to XMLHttpRequest blocked by CORS policy}
\textbf{Solution:} Check CORS configuration in backend, ensure frontend URL is whitelisted

\subsubsection{JWT Token Expiry}
\textbf{Error:} \texttt{JsonWebTokenError: jwt malformed}
\textbf{Solution:} Clear localStorage and login again, or implement token refresh

\section{Security Checklist}

Before deploying to production:

\begin{itemize}
    \item Change all default passwords and secrets
    \item Enable HTTPS with valid SSL certificate
    \item Configure firewall rules
    \item Set up database backups
    \item Enable database authentication
    \item Configure environment-specific variables
    \item Run security audit tools
    \item Enable logging and monitoring
    \item Implement rate limiting
    \item Set up DDoS protection
    \item Configure secure headers
    \item Enable API authentication
    \item Test SQL injection and XSS protection
    \item Verify CORS configuration
\end{itemize}
